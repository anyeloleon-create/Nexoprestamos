# NexoPréstamos — Gestor Inteligente de Préstamos

> *"Porque prestar no debería ser un acto de fe… sino de datos."*
## Estructura del repositorio

```
NexoPréstamos/
├── src/                   ← Código fuente (algoritmo)
│   ├── main.py            ← Punto de entrada, menú principal
│   ├── clsUsuarios.py     ← Clase y lógica de usuarios
│   ├── clsPrestamo.py     ← Clase préstamos, ítems, lógica difusa
│   ├── documentos.py      ← Generación de certificados y facturas
│   └── administrador.py   ← Panel de administración
├── doc/                   ← Documentación
│   └── MANUAL_USUARIO.md  ← Manual de uso completo
├── VERSIONADO.md          ← Plan de versionado
├── -NexoPrestamos_README -.md   ← README completo del proyecto
├── -NexoPrestamos_Actas-.md     ← Actas del equipo
└── README.md              ← Este archivo
```

## Ejecución rápida

```bash
cd src
python main.py
```
